## General information

This repository contains part of the simulation code for [Migrant Resettlement by Evolutionary Multi-objective Optimization](https://arxiv.org/abs/2310.08896).

## Requirements

```
- Python 3.6
- Igraph
- Seaborn
- Pandas
```

## Environments

```
- 3.7 GHz Intel Core i9 10900k processor
- 32 GB RAM
```
