Read me
=======
This repository contains part of the simulation code for the Migrant Resettlement Problem.


Requirements
------------
- Python 3.6
- Igraph
- Seaborn
- Pandas

Environments
------------
- 3.7 GHz Intel Core i9 10900k processor
- 32 GB RAM
